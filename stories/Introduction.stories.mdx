# Buildkite Docs Storybook

All of [Docs](https://buildkite.com/docs)' styles and components in one place, thanks to [Storybook](https://storybook.js.org/)!

## Getting started

1. Install dependencies `npm install`
2. Spin up `npm run storybook`
3. Visit `http://localhost:6006/`
